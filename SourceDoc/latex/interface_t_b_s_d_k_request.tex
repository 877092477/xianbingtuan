\subsection{Detailed Description}
T\+B\+S\+D\+K的真正的网络请求类

此类主要做联网控制，如超时时间、最大联网个数、取消所有网络连接等。 T\+B\+S\+D\+K的真正的网络请求类，此类通过delegate来获取网络请求 的配置参数（get或post，post的value，需要上传的数据）。
