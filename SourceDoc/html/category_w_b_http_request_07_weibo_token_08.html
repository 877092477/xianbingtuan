withCompletionHandler: (WBRequestHandler)  handler 

使用RefreshToken去换取新的身份凭证AccessToken.

在SSO授权登录后，服务器会下发有效期为7天的refreshToken以及有效期为1天的AccessToken。 当有效期为1天的AccessToken过期时，可以调用该接口带着refreshToken信息区换取新的AccessToken。

Parameters
refreshTokenrefreshToken
queue指定发送请求的NSOperationQueue，如果这个参数为nil，则请求会发送在MainQueue( [NSOperationQueue mainQueue] )中。
handler完成请求后会回调handler，处理完成请求后的逻辑。
