Detailed Description

The object that acts as the delegate of the empty datasets. The delegate can adopt the DZNEmptyDataSetDelegate protocol. The delegate is not retained. All delegate methods are optional.

All delegate methods are optional. Use this delegate for receiving action callbacks.
