Detailed Description

微信通知第三方程序，要求第三方程序显示的消息结构体。

微信需要通知第三方程序显示或处理某些内容时，会向第三方程序发送ShowMessageFromWXReq消息结构体。 第三方程序处理完内容后调用sendResp向微信终端发送ShowMessageFromWXResp。
