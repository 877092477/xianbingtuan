\subsection{Detailed Description}
接收并处理来自微信终端程序的事件消息 

接收并处理来自微信终端程序的事件消息，期间微信界面会切换到第三方应用程序。 \mbox{\hyperlink{class_w_x_api_delegate-p}{W\+X\+Api\+Delegate}} 会在handle\+Open\+U\+R\+L\+:delegate\+:中使用并触发。
