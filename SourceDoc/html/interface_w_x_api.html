WXApi的成员函数，向微信终端程序注册第三方应用。

需要在每次启动第三方应用程序时调用。第一次调用后，会在微信的可用应用列表中出现。 iOS7及以上系统需要调起一次微信才会出现在微信的可用应用列表中。
