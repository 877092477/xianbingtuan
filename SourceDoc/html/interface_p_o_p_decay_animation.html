◆ originalVelocity

- (id) originalVelocity
readnonatomiccopy

The original velocity value. Since the velocity property is modified as the animation progresses, this property stores the original, passed in velocity to support autoreverse and repeatCount.

◆ velocity

- (id) velocity
readwritenonatomiccopy

The current velocity value. Set before animation start to account for initial velocity. Expressed in change of value units per second. The only POPValueTypes supported for velocity are: kPOPValuePoint, kPOPValueInteger, kPOPValueFloat, kPOPValueRect, and kPOPValueSize.
