Detailed Description

视频URL对象 用于分享目标内容为视频的URL的对象

QQApiVideoObject类型的分享，目前在Android和PC QQ上接收消息时，展现有待完善，待手机QQ版本以后更新支持 目前如果要分享视频，推荐使用 QQApiNewsObject 类型
